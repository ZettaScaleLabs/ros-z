# Introduction

Welcome to the ros-z documentation! This book provides comprehensive guides and examples for using ros-z, a native Rust ROS 2 implementation built on Zenoh.

## What is ros-z?

ros-z is a high-performance, native Rust implementation of ROS 2 that leverages [Zenoh](https://zenoh.io/) for communication. It provides:

- **Native Rust**: Full type safety and zero-cost abstractions
- **Zenoh-powered**: High-performance, flexible communication
- **ROS 2 compatible**: Works seamlessly with existing ROS 2 systems
- **Modern API**: Ergonomic and idiomatic Rust interface

## Features

- Publishers and subscribers with full ROS 2 message support
- Services (request/response pattern)
- Actions (long-running tasks with feedback)
- Custom message generation
- Integration with standard ROS 2 tools
